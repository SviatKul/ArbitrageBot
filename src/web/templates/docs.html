Инструкции, математика и где брать ключи

1
Клонируйте и настройте окружение
cd ArbitrageBot
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
2
Заполните .env
Скопируйте .env.example в .env и укажите ключи хотя бы для двух бирж. Без кредентиалов бот запускается в режиме DRY_RUN=true — сделки не исполняются.
3
Запустите веб-интерфейс
python web.py
Откройте http://localhost:8080 в браузере. При первом запуске вы попадёте на страницу создания аккаунта.
4
Запустите бота
Нажмите кнопку ▶ Запустить на дашборде — или из терминала:
python run.py
5
Бэктестинг
Проверьте текущее состояние рынка без исполнения сделок:
python backtest.py --live --stake 100
python backtest.py --csv data/quotes_history.csv --output results.json

Суть арбитража на рынках предсказаний

Каждый исход бинарного события торгуется как контракт в диапазоне $0–$1. При наступлении события YES = $1, NO = $0 (и наоборот). В норме: YES_ask + NO_ask ≈ $1.

Арбитраж возникает когда на разных биржах сумма цен ниже $1 с учётом комиссий:

Прибыль = $1 − YES_askA × (1 + fee_A) − NO_askB × (1 + fee_B)

Торговля выгодна, если Прибыль > 0:

YES_askA + NO_askB < 1 − fee_A × YES_askA − fee_B × NO_askB

Пример: YES @ Polymarket = 0.44, NO @ Kalshi = 0.44, fee = 2% + 7%

Прибыль = 1 − 0.44×1.02 − 0.44×1.07 = 1 − 0.449 − 0.471 = +0.08 (+8.0%)
Критерий Келли

Оптимальный размер позиции по частичному Келли:

f* = edge / (1 − edge)
stake = kelly_fraction × f* × bankroll

Где edge = profit_percent / 100. По умолчанию kelly_fraction = 0.25 (четверть-Келли).

Polymarket — публичные данные бесплатны
Для live-торговли: docs.polymarket.com
  1. Создайте кошелёк через приложение Polymarket (Web3)
  2. Получите API Key/Secret/Passphrase через POST /auth/api-key
  3. Экспортируйте Private Key вашего кошелька
  4. Установите py-clob-client (уже в requirements.txt)
Цены доступны без авторизации. Live-ордера требуют Polygon-кошелёк.
Kalshi
kalshi.com/account/api
  1. Зарегистрируйтесь на Kalshi (требуется верификация)
  2. Перейдите в Account → API
  3. Создайте API Key → скопируйте Key ID
  4. Скачайте Private Key PEM файл (RSA)
  5. Укажите путь к PEM в KALSHI_PRIVATE_KEY_PATH или вставьте содержимое в KALSHI_PRIVATE_KEY_PEM
Kalshi работает только в США. Taker fee: 7%.
Betfair
developer.betfair.com
  1. Создайте аккаунт Betfair (верификация личности)
  2. Перейдите в Developer Portal → Create App Key
  3. Укажите BETFAIR_USERNAME, BETFAIR_PASSWORD, BETFAIR_APP_KEY
Betfair — крупнейшая P2P биржа. Комиссия: 5% с выигрыша (Premium Charge может быть выше).
Smarkets
developers.smarkets.com
  1. Создайте аккаунт Smarkets
  2. Перейдите в API → Generate Token
  3. Укажите SMARKETS_API_TOKEN в .env
Самая низкая комиссия среди UK-бирж: 2% с выигрыша.
Betdaq
developer.betdaq.com
  1. Зарегистрируйтесь на Betdaq
  2. Запросите API доступ на developer.betdaq.com
  3. Укажите BETDAQ_USERNAME, BETDAQ_PASSWORD, BETDAQ_API_KEY
Комиссия: 2% с выигрыша.
Matchbook
matchbook.com/api
  1. Зарегистрируйтесь на Matchbook
  2. Укажите MATCHBOOK_USERNAME и MATCHBOOK_PASSWORD в .env
  3. Bот авторизуется автоматически при первом запросе
Самая низкая комиссия: 1.5% с выигрыша.
Telegram Bot
  1. Напишите @BotFather команду /newbot
  2. Следуйте инструкциям, скопируйте Bot Token
  3. Напишите вашему боту любое сообщение
  4. Перейдите по ссылке https://api.telegram.org/bot<TOKEN>/getUpdates?allowed_updates=["message"]
  5. В ответе найдите "chat": {"id": 123456789} — это ваш Chat ID
  6. Укажите TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID в .env
