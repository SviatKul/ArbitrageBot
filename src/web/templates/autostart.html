Автозапуск при старте системы

macOS LaunchAgent — дашборд и Cloudflare Tunnel стартуют автоматически при входе в систему.

Оба файла plist устанавливаются для текущего пользователя. Сервисы перезапускаются при падении (KeepAlive).
Часть 1 — Веб-дашборд
1
Сохрани plist дашборда
Файл запишется в ~/Library/LaunchAgents/
2
Активируй дашборд
Выполни в терминале:
launchctl load {{ plist_path }}
Часть 2 — Cloudflare Tunnel (публичный доступ)
3
Сохрани plist туннеля
URL будет записан в лог logs/tunnel.log
4
Активируй туннель
launchctl load {{ tunnel_path }}
После запуска URL появится в логе: grep trycloudflare {{ root }}/logs/tunnel.log
5
Проверь что оба сервиса работают
launchctl list | grep arbitrage
Должны появиться строки с com.arbitragebot.web и com.arbitragebot.tunnel.
Управление
Остановить дашборд:
launchctl unload {{ plist_path }}
Остановить туннель:
launchctl unload {{ tunnel_path }}
Удалить оба из автозапуска:
rm {{ plist_path }} {{ tunnel_path }}
