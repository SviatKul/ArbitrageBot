Автозапуск арбитражного бота

Бот запускается локально на твоём Mac и автоматически стартует при входе в систему.

Дашборд уже в облаке Online 24/7
Работает на Railway — автозапуск не нужен. Доступен по постоянной ссылке.
Бот (арбитражный движок) запускается отдельно на твоём Mac и требует API-ключи бирж.
1
Убедись что есть API-ключи
Бот требует минимум 2 биржи. Добавь ключи в Settings → Мои ключи API.
Минимальный набор для старта:
  • Polymarket — API Key (уже настроен)
  • Kalshi — API Key ID + Private Key PEM
2
Проверь запуск вручную
Сначала убедись что бот стартует без ошибок:
{{ python_bin }} {{ root }}/run.py
Бот должен написать Exchange enabled: Polymarket и начать сканировать рынки. Остановить: Ctrl+C
3
Сохрани файл автозапуска
Установит бот как LaunchAgent — будет стартовать при входе и перезапускаться при падении.
4
Активируй автозапуск
Выполни в терминале:
launchctl load {{ plist_path }}
5
Проверь что бот работает
launchctl list | grep arbitragebot
Должна появиться строка с com.arbitragebot.bot. Логи: {{ root }}/logs/bot.log
Управление
Остановить бот:
launchctl unload {{ plist_path }}
Удалить из автозапуска:
launchctl unload {{ plist_path }} && rm {{ plist_path }}
Смотреть логи в реальном времени:
tail -f {{ root }}/logs/bot.log
